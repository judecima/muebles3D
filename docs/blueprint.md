# **App Name**: MuebleCAD 3D

## Core Features:

- Selector de Muebles Paramétricos: Permite al usuario elegir entre una variedad de muebles de MDF predefinidos (placard, escritorio, etc.) para su personalización.
- Configuración de Dimensiones Interactivas: El usuario puede ingresar y ajustar el alto, ancho, profundidad y otras medidas del mueble a través de campos en el panel lateral, actualizando el modelo 3D en tiempo real.
- Visualizador 3D Interactivo y Rotable: Muestra el mueble en un entorno 3D, permitiendo al usuario rotarlo, hacer zoom y visualizarlo desde cualquier ángulo mediante controles intuitivos.
- Animación de Puertas y Cajones: Funcionalidad para abrir y cerrar puertas y deslizar cajones en el modelo 3D, reflejando su comportamiento real.
- Vista de Despiece (Explosionada): Una herramienta que separa visualmente cada pieza del mueble en el modelo 3D, facilitando la comprensión de su construcción y despiece.
- Generación Automática de Lista de Corte (Despiece): Muestra una tabla detallada con el nombre de cada pieza, sus dimensiones (alto, ancho) y la cantidad necesaria, actualizándose dinámicamente con los cambios del mueble.
- Herramienta de Optimización de Corte con IA: Un tool inteligente que, basándose en la lista de corte generada, sugiere la disposición más eficiente de las piezas en paneles de MDF estándar para minimizar desperdicios.

## Style Guidelines:

- Esquema de color general claro. Primario: Un azul-gris profesional y técnico para elementos interactivos y encabezados (#3A7296). Fondo: Un blanco muy suave con un tinte azul-gris para un ambiente limpio y enfocado (#F3F6F8). Acento: Un turquesa claro y vibrante para destacar acciones clave y notificaciones (#66DBDB).
- La fuente 'Inter' (sans-serif) para todo el texto. Su estilo moderno y objetivo es ideal para presentar datos técnicos y garantizar una legibilidad óptima en parámetros y listas de corte.
- Uso de íconos lineales y geométricos, limpios y sencillos. Deben ser directamente representativos de sus acciones (rotar, abrir puerta, explocionar) o de los tipos de muebles.
- Un diseño claro de dos columnas, con un panel de control lateral a la izquierda que alberga todos los selectores y entradas, y una gran área principal a la derecha para la visualización 3D interactiva.
- Transiciones suaves y fluidas para la apertura y cierre de puertas/cajones. Animaciones sutiles para la generación y destrucción de modelos 3D, y un movimiento delicado al activar la vista explosionada para no distraer al usuario.
